# Arctic Monkeys - Pagina Tributo
Sitio web tributo de Arctic Monkeys con Django.

Deploy funcional en Heroku

## Cambiar Secret Key

Luego de instalar correr:

```python manage.py djecrety -d arcticMonkeys```

### Requisitos

```asgiref==3.2.10
dj-database-url==0.5.0
Django==3.1
django-heroku==0.3.1
djecrety==1.0.13
gunicorn==20.0.4
pytz==2020.1
